# FrameTimer
FrameCap é um otimizador de sistema em baixo nível, desenvolvido em C++ e compilado via w64devkit. Ele foi projetado para extrair performance máxima de processadores Intel (como o i3-3240) e GPUs com VRAM limitada, atuando diretamente no Kernel do Windows e nas APIs gráficas para eliminar gargalos de memória e latência
